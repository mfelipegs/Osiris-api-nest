## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Documentation

To view the API documentation, run `npm run start:dev` and navigate to [http://localhost:3000/api](http://localhost:3000/api/docs) to access the Swagger UI.
